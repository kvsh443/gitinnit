# gitinnit
Git First Setup on Dev Envs

### Windows (PowerShell)
```powershell
irm https://gitinnit.pages.dev/win | iex
```


### Linux (Headless Ubuntu / Debian Bash)
```bash
curl -sSL https://gitinnit.pages.dev/ubuntu | bash
```

## Repository Structure

`_worker.js` is the Cloudflare Pages routing script

`setup.ps1` is the PowerShell script

`setup.sh` is the Bash script
